Question 6 : Define Data View, Report View, and Model View. Explain the purpose of each view.

The Data View allows you to inspect the actual records within your tables after they have been loaded from Power Query. It is used for data validation, creating columns and measures using DAX functions.

The Report View is the default canvas where you design the visual part of your project. A drag-and-drop workspace where you add charts, maps, and slicers to create interactive and meaningful dashboards.

The Model View is one of the three main interfaces in Power BI Desktop. It provides a visual diagram of your entire data architecture.
